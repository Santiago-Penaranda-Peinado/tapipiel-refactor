# Imágenes del sitio

Coloca aquí todas las imágenes del proyecto:

## Estructura recomendada:

```
images/
├── logo/
│   ├── logo.png
│   ├── logo-white.png
│   └── favicon.ico
├── services/
│   ├── tapiceria.jpg
│   ├── limpieza.jpg
│   └── restauracion.jpg
├── gallery/
│   ├── trabajo-1.jpg
│   ├── trabajo-2.jpg
│   └── ...
└── icons/
    ├── whatsapp.svg
    ├── phone.svg
    └── ...
```

## Optimización:

- Usa WebP para mejor compresión
- Mantén imágenes bajo 500KB
- Usa lazy loading en el HTML
